## 📂 **Getting Started**

### 🔧 Prerequisites
- Python 3.8+ 🐍
- Required libraries listed in `requirements.txt` 📋

### ⚙️ Installation
1. Clone the repo:  
   ```bash
   git clone https://github.com/Cyptx/AI-BTC-Price-Prediction.git
   cd AI-BTC-Price-Prediction
   ```
2. Install dependencies:  
   ```bash
   pip install -r requirements.txt
   ```
3. Run the project:  
   ```bash
   python main.py
   ```
